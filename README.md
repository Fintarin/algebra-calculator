# Fintamath

Fintamath is an application for solving mathematical problems.

## Completed

* BigInteger
* Fraction
* x + y, x - y, x * y, x / y, x ^ y
* -x
* e, pi
* abs(x)
* sqrt(x), exp(x)
* log(x, y), ln(x), lg(x), lb(x)
* sin(x), cos(x), tan(x), cot(x)
* asin(x), acos(x), atan(x), acot(x)
* x!, x!!
